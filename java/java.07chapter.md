## 배운 것
2차원 배열 정의

## 왜 중요한가?
2차원 배열 선언을 잘못하면 컴파일시 에러가 발생

## 예시 (선택)
<2차원 배열 오류>
twoDim = new int[][] -> x
twoDim = new int[][2] -> x

<2차원 배열 가능>
twoDim = new int[2][] 

twoDim[0] = new int[3];
twoDim[1] = new int[2];
