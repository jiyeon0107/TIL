## 왜 중요한가?
자바에는 크게 두 가지 자료형이 있다.
1. 기본 자료형
2. 참조 자료형
참조 자료형 중에서 딱하나 초기화할 때 예외적인 것이 있다.
String만이, 참조 자료형이지만 new를 사용해서 객체를 생성하지 않아도 되는 유일한 타입이라고 알고 있으면 된다.

## 예시 (선택)
